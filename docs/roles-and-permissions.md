# ROLES

## TECHNICAL SPECIALIST

Can:

* view devices
* create devices
* create cycles

Cannot:

* edit existing devices
* edit existing cycles
* hand over cycles
* manage users
* view audit log

---

## ADMIN

Can:

* approve users
* manage users
* write off devices
* view audit log
* modify any data

---

## GUEST

Can:

* view dashboard
* view devices
* view device history

Cannot:

* create devices
* create cycles
* edit existing records
* manage users
* view audit log
